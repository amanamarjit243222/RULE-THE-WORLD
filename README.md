# 🌍 RULE THE WORLD

![Game Banner](assets/game_banner_hero.png)

> **Live Demo:** [Play the Geopolitical Strategy Simulator](https://ruletheworldmadebyaaj.netlify.app/)

A hyperrealistic geopolitical strategy simulator set in the year 2029. Take command of a nation and navigate complex international relations, economic challenges, and military conflicts in a modern world scenario.

## ✨ Features
- **Geopolitical Strategy**: Manage your nation's resources, military, and diplomatic relations.
- **Dynamic Scenarios**: Face realistic events inspired by contemporary geopolitics.
- **Interactive UI**: A sleek, immersive interface for managing your global empire.

## 🛠️ Tech Stack
- **Frontend**: HTML5, Vanilla JavaScript, CSS3
- **Deployment**: Netlify

## 🚀 Getting Started

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/amanamarjit243222/RULE-THE-WORLD.git
   ```
2. Navigate to the project directory:
   ```bash
   cd RULE-THE-WORLD
   ```
3. Use a local server to serve the `index.html` file (e.g., Live Server extension in VS Code, or Python's `http.server`):
   ```bash
   npx serve ./
   ```
   *or*
   ```bash
   python -m http.server 8000
   ```
4. Open `http://localhost:8000` in your web browser.
